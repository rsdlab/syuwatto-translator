## ビルド
1.SPRESENSEのArduinoプロジェクトがビルドできる状態にします(詳細はSonyのSPRESENSE公式ページをご覧ください)。

2.ArudinoIDEでcompleted formフォルダにある〇〇.inoを開いてください。

3.SPRESENSEをPCに接続します。

4.〇〇.inoをマイコンに書き込みます。
